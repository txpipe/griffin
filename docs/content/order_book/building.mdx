The policy and validator script can be built with aiken v1.0.26-alpha, running `aiken build` in the `wallet/src/eutxo_examples/order_book/contract` directory. This will create/update the `plutus.json` file, where the compiled code can be extracted and pasted into the `script.txt` file.
